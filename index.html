Welcome to MoonLight Desktop Environment.

This project aims to create a desktop environment for GNU/Linux systems, mainly for those that runs in low performance devices such as old PCs, Raspberry Pi, embedded devices and others.

It is focused in modularity in order to be lightweight and adaptable, also there will not be fancy graphics. Another important aspect of this solution is the integration with the rest of the system. We aim to provide a desktop enviroment capable of integrate the different applications in your system.
